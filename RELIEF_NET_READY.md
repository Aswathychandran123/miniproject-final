## 🚀 Quick Start

### **1. Install Dependencies:**
```bash
cd disaster-resource-frontend
npm install
```

### **2. Start Application:**
```bash
npm start
```

**That's it!** Browser opens to http://localhost:3000

- Track donations
- Rate volunteers

- Submit resource requests
- Browse available resources
- Track deliveries
- Rate volunteers

### **Volunteer (V) - Purple**

- Get rated by community
